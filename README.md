# 日影シミュレータ（ブラウザのみ / MVP）

地図上で敷地（回転可能な矩形）と建物高さを設定し、指定日時の太陽位置から日影範囲を表示するツールです。  
サーバ不要の静的アプリとして動作します。

## できること（MVP）
- MapLibre + OpenStreetMap の地図表示
- 回転矩形で建物フットプリント作成／編集
- 高さ(m)と階数（最大100階）の双方向連動
- 日付・時刻・タイムゾーン指定
- 視点（ピッチ/ベアリング）変更と周回ビュー

## 使い方
1. `docs/index.html` をブラウザで開きます。
2. 「回転矩形を描く」→ 地図上をドラッグして建物を配置。
3. 高さ/階数、日付/時刻、タイムゾーンを調整。

※ GitHub Pages で `/docs` をルートにして公開する想定です。`file://` で動作しない場合は、Pages 等の静的ホスティングで開くか、簡易サーバで配信してください（詳細は `docs/DEV.md`）。
